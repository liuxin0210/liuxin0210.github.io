一、JS解析顺序和作用域：

  • 解析顺序：
    • 定义（先找var function）
    • 执行 ( 在逐步执行 )
    • 注意:如果函数名和var定义的变量相同，var会被函数覆盖
  • 作用域：
    • 每个script是一个作用域
    • 每个函数{}是一个作用域
    • 程序每执行到一个作用域，都是按照解析顺序解析代码；
    • 作用域链:从内往外找要找函数内的变量；
