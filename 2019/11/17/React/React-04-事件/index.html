其他的事件

  • 触摸事件：onTouchCancel，onTouchEnd，onTouchMove，onTouchStart
  • 键盘事件：onKeyDown，onKeyUp， onKeyPress（前两者的组合）
  • 表单时间：onChange，onInput，onSubmit
  • 焦点事件：onFocus，onBlur
  • UI元素事件：onScroll
  • 滚动事件：onWhell（鼠标滚动）
  • 鼠标事件：onClick，onContextMenu，onDoubleClick……
